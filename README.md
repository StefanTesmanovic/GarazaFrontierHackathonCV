# GarazaFrontierHackathonCV
## Projekti
- Razvoj i evaluacija agenta zasnovanog na Reinforcement Learning-u za optimizaciju strategije u igri Tribes
- Implementacija i analiza napredne Latentne Semantičke Indeksacije (LSI) za ekstraktivno sumiranje teksta
- Sistem za detekciju i klasifikaciju simbola X i O primenom tehnika kompjuterske vizije
- Prepoznavanje i praćenje ljudi iz snimaka drona korišćenjem dubokog učenja (YOLO)
- Predviđanje povrata opklada i algoritamsko trgovanje na Polymarketu korišćenjem analize javno dostupnih podataka sa blokčejna
- Hardverska realizacija i simulacija jednostavne akustične kamere za lokalizaciju izvora zvuka pomoću beamforming-a
